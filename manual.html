1. Android/PC 如何获取 bdstoken？
复制脚本，粘贴到PC版百度网盘主页的地址栏，运行。
如果开头的 javascript: 消失，手动添加即可。

2. iOS 如何获取 bdstoken？
Safari 添加本站到书签，复制代码，编辑刚添加的书签，粘贴到书签网址处。
打开百度网盘页面，Safari 左上角 'Aa' 处选择请求电脑版网页，运行编辑好的书签。

3. 如何转存秒传链接？

填入 bdstoken、秒传链接，点击'解析链接并转存'。
显示 "errno":0 即为转存成功。

4. 运行脚本显示 bdstoken undefined 怎么办？

检查是否在PC版页面，手机版页面无法获取。
Android 请使用 Chrome 或 Via 浏览器

5. 如何生成秒传链接？

选择文件后，等待秒传链接出现。
大文件计算速度较慢，请耐心等待。


所有请求在用户端进行，本站不保存任何链接
